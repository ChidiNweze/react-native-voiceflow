## Running Locally

Use npm to install all packages:

```bash
npm install
```
Set your Voiceflow Agent's API to the environment variable 'EXPO_PUBLIC_VF_API_KEY' in a .env file:
   ```js
   EXPO_PUBLIC_VF_API_KEY= = 'YOUR_AGENTS_API_KEY'
```

Run the project using Expo:
```bash
npx expo start
```
